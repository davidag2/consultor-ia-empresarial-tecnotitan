# Consultor IA Empresarial Tecnotitan

Plataforma multilingue de diagnostico empresarial impulsada por inteligencia artificial para PYMES.

## Vision

Tecnotitan permite que empresarios conversen con un consultor virtual por voz natural en espanol, ingles y portugues. La IA entrevista al cliente por modulo, analiza sus respuestas y genera reportes ejecutivos con puntuaciones, oportunidades, riesgos y planes de accion.

## Modulos iniciales

- Ventas
- Operaciones
- Finanzas
- Marketing
- Tecnologia e IA

## Modelo comercial

- Modulo individual: 100.000 COP o equivalente internacional.
- Paquete completo: 500.000 COP o equivalente internacional.
- Pagos internacionales preparados para una futura integracion con FastSpring.

## Stack inicial propuesto

- Frontend: sitio web desplegable en Vercel.
- Base de datos: Supabase Postgres.
- IA de voz: capa futura para entrevistas multilingues.
- Pagos: capa desacoplada para iniciar manualmente y migrar a FastSpring.
